Academic Grades

This page demonstrates passing a list of dictionaries and displaying it using a Jinja2 for loop.

{% for item in grades %} {% endfor %}
# Subject Grade
{{ loop.index }} {{ item.subject }} {{ item.grade }}
How this works:

In app.py, we passed a list of dictionaries:

grade_list = [
  {'subject': 'Mathematics-3', 'grade': 'O'},
  {'subject': 'Data Structures', 'grade': 'O'},
  {'subject': 'Software Engineering', 'grade': 'A+'}
]

return render_template('grades.html', grades=grade_list)

Total Subjects: {{ grades|length }}
